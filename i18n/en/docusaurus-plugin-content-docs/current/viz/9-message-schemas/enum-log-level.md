# LogLevel

Log level

## Parent Type

`LogLevel` appears in the [`Log`](./log) data structure.

## Values

| Name    | Value | Description       |
| ------- | ----- | ----------------- |
| UNKNOWN | 0     | Unknown level     |
| DEBUG   | 1     | Debug level       |
| INFO    | 2     | Information level |
| WARNING | 3     | Warning level     |
| ERROR   | 4     | Error level       |
| FATAL   | 5     | Fatal error level |
