# NumericType

数值类型

## 父类型

`NumericType` 出现在 [`PackedElementField`](./packed-element-field) 数据结构中。

## 值

| 名称    | 值 | 描述 |
| ------- | ----- | ----------- |
| UNKNOWN | 0     | 未知类型 |
| UINT8   | 1     | 8 位无符号整数 |
| INT8    | 2     | 8 位有符号整数 |
| UINT16  | 3     | 16 位无符号整数 |
| INT16   | 4     | 16 位有符号整数 |
| UINT32  | 5     | 32 位无符号整数 |
| INT32   | 6     | 32 位有符号整数 |
| FLOAT32 | 7     | 32 位浮点数 |
| FLOAT64 | 8     | 64 位浮点数 |
