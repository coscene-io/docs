# 利用三维面板可视化你的数据

本章节将以记录中的 “elevator-08015-19-24-11.bag” 文件为例，展示使用刻行的三维面板可视化数据的步骤。

## 文件准备

1. 在项目中创建一条包含 bag 文件的记录，创建步骤详情请见[创建记录](../i/viz-3-1.png)


2. 点击记录详情页中的播放记录或文件名，进入可视化页面
