# 创建设备

## 在记录中添加设备

创建记录后，在记录详情页的「设备」栏输入「设备 ID」，点击【创建设备】

![create-device-r1](../img/create-device-r1.png)

![create-device-r2](../img/create-device-r2.png)

<br />

## 在组织中创建设备

在「组织管理」页面的「设备」分页，点击【添加设备】按钮。

![org-4](../img/org-4.png)

在「添加设备」弹框中，输入设备 ID 及其他信息后，点击【确认】。其中，设备 ID 为唯一标识，不可重复。

![org-5](../img/org-5.png)

成功添加设备。

![org-6](../img/org-6.png)
