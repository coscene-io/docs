"Panel" is a modular visualization interface that can be configured and arranged into layouts, and can also be added, edited, and dragged and dropped.

ğŸ“„ï¸ 3D Panel

The "3D Panel" is a panel that displays markers, entities, camera images, meshes, URDF models, etc., in a 3D scene. This panel provides a rich visual and interactive experience, allowing for intuitive display and manipulation of 3D objects.

ğŸ“„ï¸ Log Panel

The "Log Panel" is used to display and manage machine-generated log information. It helps users monitor, filter, and analyze various events and status updates in real-time during system operation.

ğŸ“„ï¸ Plot Panel: Visualizing Data Trends

The "Plot Panel" is a tool used for plotting and displaying data trends over time or other variables. It allows users to configure multiple data series and visualize these trends in a clear and intuitive chart format.

ğŸ“„ï¸ Image Panel

Display raw and compressed images, as well as compressed videos, with 2D annotations like text labels, circles, and points. Superimpose 3D markers for additional context.
