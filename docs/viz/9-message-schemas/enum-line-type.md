# LineType

一个枚举类型，用于指示如何解释输入点以创建线条

## 父级数据结构

`LineType` 出现在 [LinePrimitive](/) 数据结构中。

## 值

| 名称        | 值 | 描述                                     |
| ----------- | ----- | ----------------------------------------------- |
| LINE_STRIP | 0     | 连接的线段：0-1, 1-2, ..., (n-1)-n |
| LINE_LOOP  | 1     | 闭合多边形：0-1, 1-2, ..., (n-1)-n, n-0     |
| LINE_LIST  | 2     | 独立的线段：0-1, 2-3, 4-5, ...    |
