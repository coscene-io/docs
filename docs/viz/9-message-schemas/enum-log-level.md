# LogLevel

日志级别

## 父类型

`LogLevel` 出现在 [`Log`](./log) 数据结构中。

## 值

| 名称    | 值 | 描述 |
| ------- | ----- | ----------- |
| UNKNOWN | 0     | 未知级别 |
| DEBUG   | 1     | 调试级别 |
| INFO    | 2     | 信息级别 |
| WARNING | 3     | 警告级别 |
| ERROR   | 4     | 错误级别 |
| FATAL   | 5     | 致命错误级别 |
