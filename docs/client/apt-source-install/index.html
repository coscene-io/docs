Installing coScene Software via APT

Add coScene GPG key​

curl -fsSL https://apt.coscene.io/coscene.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/coscene.gpg

Add Repository to Source List​

echo "deb [signed-by=/etc/apt/trusted.gpg.d/coscene.gpg] https://apt.coscene.io $(. /etc/os-release && echo $UBUNTU_CODENAME) main stable" | sudo tee /etc/apt/sources.list.d/coscene.list

Update APT Repository​

sudo apt update

Install Software​

After updating the APT repository, you can install coScene software.

  • coStudio

    sudo apt install coStudio

  • coBridge

    sudo apt install ros-${ROS_DISTRO}-cobridge

  • coListener

    sudo apt install ros-${ROS_DISTRO}-colistener
