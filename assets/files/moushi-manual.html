1. 产品概述

1.1 产品概述

眸视机器人定位和导航系统，是一个集激光雷达、视觉（双目相机、深度相机）、超声 波、惯性测量单元（IMU）等多种传感器于一体的定位和导航系统。它融合了传感器技术、 数据处理、视觉处理、自动化控制等多种技术， 解决了机器人感知环境、定位、导航、避障 问题。 基于深度学习的机器视觉能够使得机器人检测和跟踪行人及移动目标，结合传统的图像 识别还可以帮助机器人准确实时高效的识别现场设备的状态和表计读数，有利于安保／物流 ／电力巡检机器人高效准确得完成任务。 配合巡检机器人解决方案，眸视科技提供管理监控平台以及 PC 客户端和安卓客户端， 已经针对燃气、电力、油田、化工等行业进行了针对性的开发。 眸视机器人视觉处理、定位和导航系统已经和多种轮式、履带式机器人底盘、EAI、 KOBUKI 等多种机器人小车以及深圳镭神、思岚 RPLIDAR、德国 SICK、美国 Velodyne 等多 种类型雷达适配，能够很方便应用在物流／安保／ 电力/燃气/油田/化工等行业中，相关产品已经在油田、燃气、电力、化工等进行了商业部署。 眸视机器人具有的视觉处理、定位和导航系统等特点：

  1. 模块化好，易于和现有的移动机器人系统集成；

  2. 成本低，精度高（相对误差 0.3%，绝对误差正负 1 厘米）；

  3. 支持动态环境，能够智能避障和躲避行人；

  4. 精准的远程控制，支持手机客户端、Web 客户端；

  5. 完善的管理监控平台，支持客户的云平台管理；

  6. 支持各种移动机器人底盘；

  7. 支持各种传感器及多种传感器的融合（IMU+视觉，IMU+激光雷达，激光雷达+视觉 等）；

  8. 支持商业化的嵌入式硬件，兼容性好；

  9. 提供包括底盘、嵌入式硬件、SLAM 软件、机器视觉软件、管理监控平台、Web 客户 端、安卓客户端在内的整体解决方案。

1.2 履带式机器人

  1. 支撑面积大，150mm 宽的履带接地长度与整车长度比例达到 75%，接地比压为 12KPa （满载）。适合于松软、泥泞等复杂环境作业，下陷度小，滚动阻力小，越野机动性 能好。

  2. 转向半径极小，可以实现原地转向。

  3. 履带支撑面上有履齿，不易打滑，牵引附着性能好，有利于发挥较大的牵引力。

  4. 底盘高度超过 130mm，具有超强的爬坡、越障能力。

  5. 车体外壳采用 304 不锈钢材质，坚固耐用，整体防护性能好。

1.3 轮式机器人

  1. 采用直流无刷电机+减速器+实心橡胶轮，动力强通过性好。

  2. 四轮差速转向，可以原地 360 度转向，机动灵活，转向空间小。

  3. 同侧同驱，当一侧的一轮悬空所有功率全在同侧的另一轮输出，避免悬空动力不够。

  4. 选配大容量锂电池，最长续航时间大于 4 小时，最长充电时间为 2 小时（充满电 量）。

  5. 行驶速度 0-4.5Km/h 可调，精确控制。

2. 机器人开机

2.1 机器人开机

按下图片控制面板中间的按钮。开机后按键指示灯亮，电量显示面板显示当前电量。 开机前 开机后

2.2 遥控器使用

完成机器人上电后，将遥控器中间的开关上推，打开遥控器，自动连接机器人。右侧摇 杆上下推控制前后移动，左右推控制左右移动，详见下图。 遥控器操作按键说明 注意：开始控制小车时，在遥控器上使用较小的力气，以免机器人移动太快。

2.3 遥控器高级使用

针对高级用户，遥控器其他按键说明如下：

  • 左侧绿色：油门控制，缺省为向下低位

  • 右侧绿色：方向控制，分别控制机器人前、后、左、右运动

  • 中间红色：电源开关，向上推上电;

  • 中间两侧蓝色：分别是油门和方向控制微调开关，用于控制信号均衡调整

  • 中间两侧黄色：分别是油门和方向运动补偿开关，用于控制信号均衡调整

  • 下方蓝色:控制器和机器人连接状态显示

2.4 注意事项

  1. 如果该车为非防水防尘型，禁止在潮湿雨淋，灰尘粉尘多的恶劣场合使用。工作环境 温度，应该在-20℃ ～ 55℃ 范围内。

  2. 机器人应避免接触易燃、易爆、腐蚀性液体及气体。

  3. 必须了解车体的载重、爬坡、越障等性能，并在设计参数内使用。否则将减少机器使 用寿命，发生倾覆危险。

  4. 小车启动前，应先试运转检查蓄电池电量，急停开关、仪表等能否正常工作，小车转 向是否灵活，转向手柄是否正常，制动是否正常。

  5. 启动速度不宜过快，应确保小车行进范围内无人员和障碍物，将速度由慢到快逐渐调 节，避免速度突变造成伤害或损失。

  6. 小车在爬坡以及负载前行的情况下，应尽量减小速度，防止电机过载。

4. 制图

准备工作：选择制图的初始位置并粘贴二维码，（ID 号为 6/18 以外的任何 ID 的二维 码）粘贴在与可见光摄像头同一高度处。

4.1 制图

  1. 安卓平板，连接后，在主菜单界面点“遥控功能”，建议线速度不大于 0.3m/s,角速度 不大于 0.3rad/s。通过上下左右，遥控小车到二维码的正前方。

  2. 安卓平板，返回到主菜单-》制图导航功能-》制图功能-》新建地图，进入到遥控制 图模式界面。

  3. 等待有坐标显示后，遥控小车，建议线速度为 0.3m/s,角速度 0.3rad/s，控制小车先 原地旋转一圈。在需要制图的路线内，小车走来回路。覆盖地图后，制图最后再次把小车移 动到二维码的前面，距离二维码的距离要比制图开始时远一些。

  4. 点击“遥控制图模式界面右下角的“制图”，完成制图。 制图完成之后，进入到地图编辑界面，轨迹膨胀半径输入 0.5，点“执行轨迹编辑“

4.2 地图编辑

  1. 点击连接，成功后进入制图导航功能=》地图处理界面，点击地图地图编辑

  2. 点击手动编辑

  3. 选择要编辑的地图，点击确定

  4. 通过中右侧方框点开可以选择编辑读题的颜色

  5. 选中绿色代表新增的可通行区域，红色代表新增的虚拟墙，小车不可通行区 域。(下面的数字 6 是线条的粗细程度)

  6. 编辑完后选右上角的对勾

  7. 最后点击“上传”，完成地图编辑。

  8. 编辑完生成的地图

4.3 地图备份与切换

在地图编辑界面，点备份，打开设置界面，配置参数，其中二维码 ID 需要配置为当前 使用二维码的 ID 号，其他可以是字母或者文字或符号。配置完后，点“备份”，完成地图的备 份。 注意：只有二维码 ID 可以为空，其他都不可以为空。 安卓客户端主界面-》制图导航功能-》地图处理-》地图切换，可以看到备份的地图。 选中某一地图，弹出回传界面，点“发送”，机器人中的地图切换到选中的地图。如果需 要删除已备份的地图，在回传界面点“删除”，可删除地图。

4.4 注意事项

4.4.1 制图之前的准备

  1. 制图之前应确定上一个地图是否还要，是否需要备份。

  2. 尽量在待扫区域没有人，或者人较少的时间点进行扫图。

  3. 尽量清除待扫区域中经常移动的物体。

  4. 二维码贴在经过次数比较多的地方。

  5. 对于场景中的反光，透光，吸光等物体，在条件允许的情况下，尽量在机器人 的雷达扫描平面处（雷达高度上下 4cm 范围内）贴上其他不影响雷达正常扫描的材料。 （反光率过低的物体，比如黑色吸光墙面，家具等。玻璃镜子，不锈钢等镜面反射物 体）

4.4.2 建图操作原则

建图过程使用 rviz， roslaunch rcp_slam carto_rviz.launch，查看制图效果，建图过程中 建议遵循如下的原则：

  1. 制图时，尽量叫机器人走在道路的中间，在直线路上尽量叫机器人少转弯，少贴近障碍物行走

  2. 机器人在拐角处拐弯或掉头时，不一次性完成转弯。可以转 30 度，向前走一点，再转 30 度，再向前走一点。（注意：尤其是道路粗糙度大且角速度小时）

  3. 对于直线型地图，使用 360 度激光，制图走一趟（如果制图效果不好，制图时走来回路）；使用非 360 度激光，走来会路。

  4. 操作员站在机器人的正后方。

  5. 制图开始时，在原点转一圈（360 度的和非 360 度的，建议都转 1 圈）。

  6. 第一次经过二维码时，机器人离二维码近一些，第二次遇到二维码时离二维码远一些。车经过二维码时，叫机器人看一下二维码

  7. 采用较低的速度，建议小于 0.5m/s，速度视当前平台机能而定 。

  8. 多走产生闭环的环路。

  9. 环路回到原点后，保持机器人运动，多走重合的路。不可立刻停止移动。

  10. 环路回到原点后，如地图不闭合（制图过程中观看 RVIZ，如下图），则继续行走（使用较小的行驶速度，如：0.3m/s），直到闭合为止。

  11. 对已闭合的区域，避免走老路，降低内存消耗。

4.4.3 建图结果检查

建图完成后，需要判断建好的地图是否达到要求。操作如下：

  1. 重启机器；

  2. 使用 RVIZ，命令：roslaunch rcp_slam nav_rviz.launch，查看地图；

  3. 有以下检查原则：

    1. 地图中障碍物边缘是否清晰。

    2. 地图中是否存和实际环境不一致的区域（有无闭环）。

    3. 地图中是否存在机器人行动区域内的灰色区域。

    4. 地图中是否存在后续定位时将不存在的障碍物（扫图时行走的人，移动的物体）。

    5. 是否能够保证机器人在活动区域内的任何位置，360 度的视野范围内，均已经探明。

5. 导航

在机器人控制端的主界面，制图导航模式-》导航/视觉追踪-》导航功能，等待加载地 图。加载地图后，在重定位成功前，地图是灰色的。通过此界面的的控制功能，控制机器人 到制图初始的位置，旋转机器人，可快速重定位 成功，重定位成功后，地图上不再有灰色层。 可以看到地图上机器人制图时走过的位置为白色。

5.1 导航到指定点

在机器人走过的路上（白色区域）任意一点，长按，选择“开始“，机器人可以自主导航到此点。 如果点击目标地点后, 机器人没有移动,则说明该地点由于物理限制的原因无法抵达，具 体原因有：

  1. 目标空间太小,,导致机器人无法靠近该地点;

  2. 目标点距离障碍物很近;

  3. 路障导致无法行走;

  4. 超声波停障: 超声波检测到近距离的障碍物;

  5. 机器人周围有障碍物环绕;

5.2 导航到指定坐标

在机器人控制端的主界面，制图导航模式-》导航/视觉追踪-》导航功能，重定位成 功后，点“设定目标”，设定目标点的坐标后，点“确认”，机器人自动导航到目标点。

5.3 取消导航

在导航过程中，在地图信息界面点“取消导航”，机器人立即停止移动。

7. 云平台使用

7.1 系统登录

本系统有三种用户角色，分别为普通用户、管理员和超级管理员，不同用户登录成 功后，将拥有不同的功能权限。 | 用户角色 | 用户描述 | | ---------- | ---------------------------------------------------------------------------------------------------------------------------------- | | 普通用户 | 可进行任务管理、实时监控、巡检报表、设备告警查询、巡检结果 分析、系统告警查询等模块的功能使用，以及用户设置模块个人用 户信息的修改 | | 管理员 | 在普通用户基础上，可进行用户设置模块普通用户和管理员用户信 息的维护 | | 超级管理员 | 在普通用户基础上，可进行系统设置各模块、功能的维护 | 登录页面为用户名和密码登录，及验证码。

7.1.1 系统首页

系统主界面内容为：菜单、可选择受控机器人、电量、机器人充电控制、可见光/热 红外视频、云台/车体控制、电子地图、实时消息、任务状态、巡检点巡检图表信息（最 近七天）、实时气体浓度信息、巡检报表（最近三条）。 1、 菜单：根据角色显示不同菜单，可在系统管理-角色管理中给相应角色分配菜单权 限，在系统管理-菜单管理中添加、修改、删除菜单； 2、 可选择受控机器人； 3、 机器人充电/停止充电控制，充电中显示电量信息； 4、 实时视频、实时控制查看 1.3 实时监控； 5、 电子地图（2D）:

  • 显示机器人当前位置及实时运动轨迹、地图名称

  • 巡检当前巡检点、显示历史数据、显示报警信息、隐藏/显示空闲巡检点； 6、 实时消息：

  • 巡检消息：任务中每个巡检点的数据消息：如仪表值、温度值、设备状态。

  • 告警消息：车体告警消息、程序告警消息、环境报警消息

  • 系统消息：分类汇总出在巡检过程中产生的各类消息。 7、 任务状态：当前任务名称、任务状态。 8、 机器人状态：当前机器人名称、车体状态（空闲、正在执行任务、返回充电中、正 在充电、未在线）、时间、重定位状态（未定位、定位成功）。

7.1.2 个人中心

点击用户头像显示下拉框，可选择进入个人中心和退出登录。 个人中心可修改密码，及修改个人基本信息，并且可以查看当前账号操作日志。

7.2 实时监控

实时监控用于可见光、热红外视频直播以及对当前操作的机器人进行遥控操作，包 括可见光及云台操作（转动控制，光圈、焦距、变倍操作，雨刷控制、补光灯控制 等）、机器人机体操作（移动控制、机器人速度设定等）。 1、 视频直播：具有可见光视频直播窗口、热红外视频直播窗口以及相关的直播开关按 钮。 2、 云台控制：

  • 点击方向箭头可以实现云台的抬高、降低、左转、右转等操作；

  • 点击“焦距”两侧“-”和“+”按钮，控制可见光焦距调整；

  • 点击“光圈”两侧“-”和“+”按钮，控制可见光光圈调整；

  • 点击“变倍”两侧“-”和“+”按钮，控制可见光变倍调整；

  • 点击“刷子”图标，实现可见光及热红外雨刷开启操作，再次点击图标，雨刷停 止；

  • 点击“太阳”图标，开启可见光及热红外设备补光灯，再次点击图标，补光灯关 闭。 3、 车体控制：

  • 车体控制的主要作用是控制巡检机器人移动行走，通过点击控件中的不同区域 来控制巡检机器人前进、后退、左转、右转；

  • 点击方向箭头或者使用键盘上下左右键可以实现机器人前进、倒退、左转向、 右转向操作；

  • 点击“高度”两侧“-”和“+”按钮，选择机器人颈部升降高度，点击最下方的按钮控 制机器人颈部调整到选择的高度；

  • 支持机器人行进速度设定，在移动前，设置速度值，系统默认机器人最大行进 速度为 1M/S，在发送移动指令后，即可按设定好的速度移动。

7.3 数据查询

7.3.1. 巡检报表

巡检报表查询用于汇总巡检任务完成后自动生成的巡检数据报表，支持报表查询、预览、打印及导出功能。 可按日期查询巡检任务，选中某个巡检任务，即可在右侧查看巡检报表，支持缩放 和打印，点击导出按钮，可将巡检报表导出成 word 保存在本地。

7.3.2. 巡检点

巡检点查询用于显示在巡检任务过程中各巡检点巡检数据信息，包括可见光图像、 热红外图像数据分析、仪表图像数据分析、视频、音频、历史数据显示、历史曲线显 示、报表输出等。 可按设备类型和日期，查询指定日期范围内巡检到的某类设备。下方的输入框可根 据设备名称对查询出来的结果进一步筛选。点击某个设备，可以查看设备当时的巡检信 息，包含可见光图片、热红外图片、视频和音频信息（告 警时才会有视频和音频信 息）。右侧显示设备的历史记录，下方显示设备的历史曲线。

7.3.3 环境数据

环境数据查询用于展示巡检任务过程中相应传感器采集到的温度、湿度、烟感、PM2.5 等数据曲线，点击顶部温度、温度、烟感、PM2.5,即可生对应的曲线，默认显示 温度曲线。

7.4 巡检任务

7.4.1 任务管理

任务管理功能主要用于智能巡检机器人监控平台中巡检任务的日常管理操作，包括任务新建、编辑、复制、删除、查询，任务执行、任务暂停、任务终止等。 主任务功能包括如下：

  • 点击查询按钮，可按日期、任务名称、任务类型、任务状态等条件组合查询；

  • 点击新建按钮，打开新建任务窗口；

  • 点击复制按钮，可对选中的任务进行复制，重新输入任务名称，即可创建一个 新的任务；点击编辑按钮，即可对选中的任务进行编辑，打开编辑任务窗口；

  • 点击执行按钮，可将制定的任务下发至服务端执行（仅限立即执行类任务，需 手动下发，定时和周期的任务，由服务端根据时间定时下发）； 子任务功能包括暂停或恢复，终止，删除。 点击暂停或恢复按钮，即可对正在执行或暂停的任务进行暂停或恢复操作；点击终 止任务，可提前终止正在执行的任务；点击删除任务，可对未执行过的任务进行删除。 操作逻辑如下： 在新建任务窗口中，输入任务的基本信息，勾选需要巡检的设备，最后指定任务的 执行方式，点击保存按钮，即可创建一个任务。

7.4.2 任务日历

任务日历用于智能巡检机器人监控平台中巡检任务的日历样式显示及相关操作管 理，包括任务按月、日查询及显示，单日任务执行情况显示及编辑操作。 该功能以日历形式展示出当月每天执行过的任务以及待执行的任务。对于执行过的 任务，可以查看任务报表。

  3. 机器人已完成制图功能，导航功能正常。

8.1 启动检查

打开安卓设备的相应 APP 程序，依次按照以下顺序操作：主菜单-》巡检功能-》切换 巡检，进入到巡检设置界面。 按照以下步骤配置：

  1. 在监控平台所在的电脑上查看电脑的 IP 的地址。

  2. 服务 IP 和流媒体 IP 都配置为 上一步所查询的云平台 IP 地址。

  3. 点击“开启巡检”，开启巡检成功后，服务器状态为连接成功。

  4. 把机器人遥控到制图时初始位置附近，旋转机器人，可以看到重定位状态为重定位成功。

  5. 此时平板可以锁屏，目的是省电。再使用时可以再解锁屏幕。

8.2 云平台操作

云平台系统主界面内容为：菜单、可选择受控机器人、电量、机器人充电控制、可 见光/热红外视频、云台/车体控制、电子地图、实时消息、任务状态、巡检点巡检图表 信息（最近七天）、实时气体浓度信息、巡检报表（最近 三条）。

  1. 菜单：根据角色显示不同菜单，可在系统管理-角色管理中给相应角色分配菜单权 限，在系统管理-菜单管理中添加、修改、删除菜单；

  2. 可选择受控机器人；

  3. 机器人充电/停止充电控制，充电中显示电量信息；

  4. 实时视频、实时控制查看 1.3 实时监控

  5. 电子地图（2D）:

    • 显示机器人当前位置及实时运动轨迹、地图名称

    • 巡检当前巡检点、显示历史数据、显示报警信息、隐藏/显示空闲巡检点；

  6. 实时消息：

    • 巡检消息：任务中每个巡检点的数据消息：如仪表值、温度值、设备状态。

    • 告警消息：车体告警消息、程序告警消息、环境报警消息

    • 系统消息：分类汇总出在巡检过程中产生的各类消息。

  7. 任务状态：当前任务名称、任务状态。

  8. 机器人状态：当前机器人名称、车体状态（空闲、正在执行任务、返回充电中、正 在充电、未在线）、时间、重定位状态（未定位、定位成功）。 巡检过程中的云平台系统使用，参见上一章节“云平台使用”中的“实时监控”、“数据 查询”、以及“巡检任务”。

9. 注意事项

巡检机器人经常工作在严苛的环境中，因此对机器人经常进行检查保养，不仅可以延长 机器人的使用寿命，而且可以早期发现故障隐患，早期处理，并减少维修时间和维修费用。

9.1 常规检查

  1. 机器人开始工作前应注意检查机器人电池电量，确保充电功能正常，电池充满 后再开始工作，避免电量过低造成无法完成巡检全程。

  2. 当现场需要进行生产设备维护，停止巡检机器人执行任务，处理正常关机和开 机的步骤：

    1. 机器人返回到充电桩，充满电

    2. 在监控管理平台，终止所有巡检任务

    3. 手动把机器人关机，可以进行现场作业

    4. 等需要巡检时，手动把机器人开机

    5. 在监控管理平台，打开直播，遥控机器人前进约 30~50cm，重新设置周期任务，执行巡检任务。

  3. 巡检机器人在经过泥沙、碎石、沾染油污或化学药品环境使用后应及时清洗车 轮并检查、清理卡嵌的异物，以免对轮胎寿命和底盘运动部件造成损害。

  4. 正常工作期间应至少每月检查一次机器人各部分螺栓螺母固定情况，如有松动 需及时紧固。对底盘运动部件螺栓应重点检查。全车活动铰接点、轴承以及连 接处需要及时润滑并检查工况。发现有掉漆、碰伤等情况时应及时 补漆，防止 生锈进而影响结构强度。

9.2 维护保养

  1. 机器人每行驶 100KM 应进行一次检修。

  2. 当操作人员在保养或检查机器人时，禁止其他人员操作遥控器，否则有可能会造成严重的人员伤害或设备损害。

  3. 在实施维修工作之前，应确定作业的指挥员，制定作业流程，进行有计划、有 步骤的作业。

  4. 请使用正确、合格的维修工具，不得使用损坏的、劣质的工具进行机器人的维 修作业。

  5. 当拆开机器人外壳进行维修时，不要把异物掉落在机器内部，否则可能引起机 器损坏、操作失误或其他故障。

  6. 电气系统的维护应该由具备低压电气专业资格的人员 进行。

  7. 如需对机器人进行电焊作业，需要断开外接电源，断 开电池，并由具备焊接资质的人员进行。否则将可能 造成机器人电气系统损坏。

9.3 长期储存

  1. 机器人长期停止工作时，应存放在干燥、清洁、通风的室内环境，避免长时间 处于潮湿、高热、暴晒环境中，以免设备加速老化。

  2. 停止工作前应把电池充满电后再关机存放，每月至少检查一次电池电量，低于 30%电量时应及时充电，以免电池长期处于过放电状态损坏电池。

  3. 机器人停放时，应避免停靠在斜面或者障碍物上，两侧车轮应同时受力，均匀着 地，以延长轮胎的使用寿命。

10. 快速故障排除

10.1 机器人故障排查

问题：机器人显示电量为 0，不能开机 解决方案：机器人长时间不用时，应定期充电，避免电池过放影响电池使用寿命。如果 遇到电池电量放尽的情况，请按下面方法处理：

  1. 将机器人通过推车推到充电桩附近。

  2. 将充电器接通电源，手动插到机器人充电口上（机器人保持为关机状态）；

  3. 给机器人充电 3~4 小时后，然后按开机键上电；

  4. 确认电量处于正常范围后，将充电插头插回充电桩，遥控机器人回到充电桩附近， 等待机器人正常执行任务。 问题：机器人未在线 解决方案：可能有以下几种情况：

  • 云服务器是否欠费过期，请及时续费。

  • 机器人的通信模块 5G 卡是否欠费停机，请及时续费。

  • 检查局域网是否工作正常。 问题：机器人未按照预定巡检路线行进，但最终到达巡检点 解决方案：当预定巡检路线被障碍物遮挡时，机器人会自动规划新的路线绕过障碍物。 此时应检查预定巡检路线上是否有不易察觉的障碍，如平滑反光的水面、油污、深度较大的 裂缝等干扰机器人行进的障碍。 问题：机器人在离开充电桩的位置，原地旋转 解决方案：当局部区域环境发生变化时，机器人容易在此区域附近丢失定位信息。此时 机器人会自动启动重定位功能，原地旋转，收集更多的位置信息，帮助重新匹配地图位置信 息。大部分情况下机器人通过原地旋转可以 重新定位成功。如果机器人长时间定位不成功、 或环境没有明显变化时也经常原地旋转重定位，请联系眸视技术人员协助分析解决。 问题：机器人不能正常自动充电，充电器指示灯不亮 解决方案：

  • 请检查充电桩是否通电（市电 200v，功率 5000+W）。 问题：机器人巡检过程中长时间停障，监控管理平台告警提示充电失败 解决方案：

  • 机器人是否进到障碍物里边或距离障碍物较近，无法移动，通过监控管理平台远程 反向慢慢移动机器人到空旷的巡检路线上，点击一键充电。

  • 如果电量过低，请用遥控器移动机器人到充电桩附近，关机进行手动充电，充满后 开机。（参考“问题 1”操作即可） 问题：机器人在狭窄的位置不移动 解决方案：当机器人离障碍物距离太近触发停障，并且尝试脱离障碍物不成功后，机器 人会停止行进并发出报警。 解决方法：

  1. 通过监控管理平台，遥控机器人离开此空间狭窄的地方

  2. 机器人重定位成功后如果可以继续执行任务。

  3. 如果任务已经超时，不能继续执行任务。此时可以在机器人重定位成功后，在监控 管理平台下发一键充电任务，使机器人自主返回充电桩等待下一次任务执行。 问题：登录监控管理云平台，提示 websocket 连接失败** 解决方案：检查电脑联网是否正常。 问题：执行任务机器人不动 解决方案：

  4. 确认网络连接，比如尝试控制云台

  5. 通过监控管理平台，点击车体状态确认各传感器状态，雷达、imu、里程计、摄像 头、云台、升降杆等状态是否正常

  6. 确认机器人附近没有障碍物，尽量在较空旷的区域执行任务 问题：机器人离开充电桩后 重定位时间过长导致任务超时 解决方案：此问题通常由于充电桩附件环境经常变化造成。选择充电桩位置时应考虑环 境固定、特征点突出的地方，并具有足够大的无障碍空间。如果此问题经常发生，请联系厂 家协助分析解决。

10.2 云端服务故障排查

问题：某一时刻的任务没有生成报表 解决方案：执行任务过程中，任务状态被置为了停止或取消，则此次巡检任务不会生成报表。 问题：监控管理云平台系统操作无反应 解决方案：需要执行云端服务器重启，（也可以参见 6.5 章节）

  1. 后台 java 程序启动

    1. 进入/home/eladmin/目录下，输入./stop_eladmin.sh 启动关闭程序

    2. 输入./start_eladmin.sh 为开启后台程序

  2. 前台 tomcat 启动

    1. 关闭：/home/apache-tomcat-8.5.4/bin# ./ shutdown.sh

    2. 开启：/home/apache-tomcat-8.5.4/bin# ./ startup.sh

  3. 如果重启云端服务器还无法控制，请重新把机器人关机、重启。 问题：报表中某一巡检点任务没有检测结果 解决方案：该巡检点附近有障碍物，导致机器人无法到达该巡检，巡检任务超时所致， 请及时清理巡检路线上的障碍物。

  4. 遥控器故障排查 问题：低电压报警 解决方案：如下图所示，显示遥控器电压太低（低于 11.5V），导致无法正常工作，更 换 8 节 5 号电池即可。 问题：混控报警 解决方案：如下图所示，油门杆位置没有设置到最下方，系统提示”警告混控已经打 开”，并持续红灯闪烁提示告警，需要将左侧油门杆拉到最下。 问题：遥控器控制失灵 解决方案：调节油门和方向微调开关和运动补偿开关，确保都设置于中位。 问题：其他报警 解决方案：遥控器故障，需要联系原厂技术服务支持。

附录 1：充电桩安装说明

  1. 用于充电的二维码 id 固定为 6 和 18（左 6 右 18），二维码边长固定为 0.1 米

  2. 二维码高度：与可见光相机在同一个高度

  3. id6 的二维码位置： 首先需要确定可见光相机相对机器人中心点左右的偏移。小车坐标系如下所示，x 轴指 向机器人正前方： 若可见光相机在 y 轴上的值为 0，那么该二维码应该贴在充电桩正上方; 若可见光相机在 y 轴上的值不为 0,那么该二维码应该贴在充电桩正上方偏移距离 y 的地 方，（y 正应该向左侧偏，y 负应该向右侧偏）。

  4. id6 位置验证： 机器人在充电桩充电时应确保可见光相机正对 id6 的二维码

  5. id18 的二维码位置： 与 id6 同一高度，在 id6 右侧 0.31 米处（该位置需精确到 0.1 厘米）；

  6. 对接上后 二维码与可见光的距离应确保>=0.2 米（计算方式是云台向后看时的距离）。

  7. 添加充电桩前方巡检点位置： 巡检点距离充电桩的距离应稍大于： 4x 机器人半径。

附录 2、传感器清洁

为了能够准确地感知周围环境,机器人配置的各个传感器需要保持洁净,特别是激光雷达 的环形防护罩。

2.1 注意事项

请在清理激光雷达以及各个传感器之前，仔细并完整的阅读本附录的内容,否则不当的 操作可能会损坏设备。 由于激光雷达和各个传感器需要在环境比较恶劣的情况下使用,需要及时清理表面的脏 污保持雷达清洁,否则会影响设备的正常工作。

  1. 需要的材料

    1. 洁净的纤维布

    2. 装有中性的温洗皂液的喷雾

    3. 装有洁净的水的喷雾

    4. 异丙醇溶剂

    5. 干净的手套

  2. 清洁方法 如果雷达的表面只是粘附了一些灰尘/粉尘,可直接用洁净的纤维布粘少量的异丙醇溶液, 然后轻轻地对雷达表面拭擦清洁,再用一块干燥洁净的纤维布将其擦干。 如果雷达表面沾上了泥浆等块状异物,首先应使用洁净水喷洒在雷达脏污部位表面让泥 浆等异物脱离(注意:不能直接用纤维布将泥浆擦掉 ,这样做可能会划伤表面特别是防护罩表 面)。其次用温的肥皂水喷洒在脏污部位,因肥皂 水的润滑作用可加速异物的脱离。再次用纤 维布轻轻试擦雷达表面,但注意不要擦伤表面。最后用洁净的水清洗雷达表面肥皂的残留(如 果表面仍有残留,可用异丙醇溶液对其再次清洁 ),同时用一块干燥的微纤微布擦干。
